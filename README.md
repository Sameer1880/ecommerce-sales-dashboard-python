# E-commerce Sales Dashboard

## Project Overview
This project is an interactive **E-commerce Sales Dashboard** built with Python to analyze and visualize sales data. It provides insights into revenue trends, order counts, and top-selling products to support data-driven business decisions.

## Features
- Analyze total sales, revenue trends, and number of orders
- Identify top-selling products and categories
- Visualize data with interactive charts
- Track business performance over time

## Key Metrics
- Total Revenue
- Number of Orders
- Product Sales Performance
- Sales Trends over Time

## Tools & Technologies
- **Python** – Core programming language
- **Pandas** – Data cleaning and analysis
- **NumPy** – Numerical calculations
- **Matplotlib / Seaborn / Plotly** – Data visualization
- **Jupyter Notebook** – Development and interactive analysis

## Skills Applied
- Data Analysis
- Data Visualization
- Dashboard Development
- Marketing & Sales Analytics
- Business Intelligence
